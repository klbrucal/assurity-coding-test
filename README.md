# Coding Test - Assurity

This is a test project that will validate the following Acceptance Criteria:

1. Name = "Home & garden"
2. CanRelist = "true"
3. The Promotions element with Name = "Feature" has a Description that contains the text "Better position in category"

## Requirements

1. Java installed
2. Maven installed
3. Add path to environment variables

## How to run (Windows)

1. Clone the repository
2. Run CMD
3. type cd [PROJECT_FOLDER] (locate where you clone the project)
4. type mvn clean install
5. type cd target
6. type index.html (to view the results)
